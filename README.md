# Aeron Playground


## Aeron Pub

Build:
`./gradlew jar`

To run with embedded driver:
`java -jar -Daeron.sample.embeddedMediaDriver=true ./build/libs/aeron-pub-1.0-SNAPSHOT.jar`


## Aeron Sub

Build:
`./gradlew jar`

To run with embedded driver:
`java -jar -Daeron.sample.embeddedMediaDriver=true ./build/libs/aeron-sub-1.0-SNAPSHOT.jar`
